# StyleMatch

GitHub Pages-ready static website.

## How to deploy

1. Upload `index.html` and `README.md` to the root of your GitHub repository.
2. Go to Settings -> Pages.
3. Source: Deploy from a branch.
4. Branch: main, folder: /(root).
5. Open `https://YOUR_USERNAME.github.io/YOUR_REPO/`.

The site is a browser-only demo using localStorage and public Wikimedia image URLs.
